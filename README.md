O Playfisica é uma plataforma educacional interativa projetada para ensinar conceitos fundamentais de Física através da lógica de programação. O objetivo é transformar o aprendizado passivo em uma experiência ativa, onde o aluno visualiza fenômenos físicos em tempo real ao manipular código e parâmetros.

- Tecnologias e Ferramentas
Linguagem: Kotlin

UI Framework: Jetpack Compose (Interface reativa e moderna)

Backend: Firebase

Authentication: Gestão de usuários e progresso.

Firestore/Realtime DB: Armazenamento de lições e scripts.

Arquitetura: MVVM (Model-View-ViewModel) para uma separação clara de responsabilidades.
